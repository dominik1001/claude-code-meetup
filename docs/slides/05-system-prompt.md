# Slide 05 — The System Prompt

## Position in Arc
First concept. Answers the question: if it knows nothing, what does it start with?

## Visual
- Background: near black `#080808`
- Envelope illustration — center, dark brown with gold border
- Envelope flap open at top with a fold line
- Dark red wax seal with "A" on the flap — Anthropic's instructions

## Text

**Title:** THE SYSTEM PROMPT
**Subtitle:** *The letter it arrives with. Before you type a single word.*

## Spoken Words
> "So if Claude wakes up every time with no memory — what does it actually know when it starts?"

> "It arrives with a letter. A set of instructions written by Anthropic, baked in before you ever open the terminal. That's the system prompt. It tells Claude who it is, how to behave, what tools it has access to. You didn't write it. You can't see it. But it shapes everything."

> "Think of it as the job description handed to someone before their first day."

## Timing
~1.5 minutes

## Design Notes
- Title in Georgia bold, 40pt, cream — wide letter spacing (charSpacing: 5) for gravitas
- Subtitle in Calibri italic, 19pt, grey — explanatory, not decorative
- The envelope is pure illustration — dark, lit from the side, the seal the only accent of color
- The "A" on the wax seal is a subtle nod to Anthropic

## Why This Slide Exists
The system prompt is invisible to most users but foundational to understanding Claude's behavior.
The envelope metaphor makes it tangible: instructions that arrive before the conversation begins.
It also sets up the context window — the envelope goes *into* the context window.
