Summarize the three learnings from the presentation:

1. **The Harness** — Claude Code is only a small harness on top of an LLM. The magic is in the model.
2. **The System Prompt** — Use the right tool for the task. Adapt the prompt to the task at hand.
3. **The Agentic Loop** — Think what the model can do best. Give it the right tools.

Tie back to the core message: "Any technology you understand is no longer magic."
